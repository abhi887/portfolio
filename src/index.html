Portfolio

Works

All Python Java Python - Flask Python - Django Reactjs Nodejs + Reactjs PWA

Bunk-inator

A simple and smart, screen capture desktop environment software application which stores entire journey from the start of application session to the end in the form a series of screenshots. while trying to avoid any redundancies by eliminating similar images.

ascii-filters

A python application which can convert any image, gif video or any other normal video to it's ascii-version. in case of video it can also do real-time image to ascii-art conversion making the system feel more reactive.

qr2go

qr2go was created as part of a bigger project with the intention of solving a problem of lost packages between order and delivery. qr2go will act as a platform which will keep track of all the products in the system right now as well create qr code labels for the products as they are dispatched for delivery.

PMA (Powdery Milkdew Risk Assesment)

PMA or Powdery Milkdew Risk Assesment is a risk assesment technique used in agriculture to find whether certain crops are affected by flu or not. It was a final year thesis project for a PhD student which involved collecting data from a several IOT nodes and accumulating that in one place with a complicated algorithm in place to calculate risk associated with data and interactive visual data assessment.

Naiee

Naiee is a simple instant messaging application which will require not login or no user data of any kind except for the username that is given to it by the user and the user can start messaging all the available users on the platform. it uses the concept of closed groups chatting, so the users chatting will have to be in the same channel or group.

Readers Groove

Readers Groove is a website for reading reviews and detailed information about all the book on the website, it served as learning project along the way in the very beginning of the journey when i first started learning web developement.

abhizza

abhizza's is a food ordering application which has integreation with Stripe api as a means of payments gateway and processing. later on qr2go was also integrated with abhizza to create a complete application process from food order to delivery.

joshis

joshis was another commercial project undertaken for a local group of english classes, the application has various capabilities such as regurlarly updating tuition notices, posting unlimited number of blogs, a unique sharable link for each blog post etc.

slamme

slamme was a summer hobby project which started as an idea among the group of friends about how we used to get each of us to sign our slam books, and taking that idea and making it digital. So in the end that's was it turned out to be, it was a college level social network where all the students of our college will go to and send slam requests and write slams to other people, it was also so popular to be used on mobile, i never developed the desktop side styling of it.

Modern's E-library

Modern's E-library is as the name says is a digital library primary aimed at the students of modern college, but as the application's database grew and its features grew too, it was not ethical to reserve it for one college only, so i released the application for all on the internet with anyone who has access to it can read or download any book they like. as of today it has close to 2500 community donated books in its database.

encrypter

encrypter stands apart from the crowd here as its does not have a nice User Interface or any at all. That is because enctypter is a CLI application as the work of enctypter is to encrypt or decrypt files and folders in the computer and a end user will rarely need to have that kind of functionality so the people who needed that kind of functionality will already be equipped with knowladge about what to do with it.

voice-notes

Voice-notes is a single page application with the purpose of increasing your productivity by helping you quickly take notes in the form of voice notes/memos. All the data stored by voice notes always stays on your device so there is also no fear for privacy voilations.

todo-lists

Todo-lists is a single page application with the purpose of increasing your productivity by helping you quickly keep track of all the todo's you currently have on the task list. it also provides a very minimalistic UI to maximize the productivity and is developed as part of PWA to give instant access on multiple devices.

Proton-editor

proton editor is a web based code editor with linting and text highlighting capability and support for more than 50 languages with more than 30 themes. It also supports real-time code collaboration with others by using just the name of the code editor, as the proton is just a prefix of any name given by user to the editor.

clipboard

clipboard is shared clipboard with access across any number of devices to the same clipboard. It works by sharing and using the same name of clipboard across multiple devices, with server only having a in-memory nosql database. providing for low latency updated database access and realtime sharing of clipboard.

todo-list PWA

todo-list as a pwa is exactly similar as the todo-list webapp but with a significant distinction being that, todo-list's pwa is a installable application available on android and windows systems using the same codebase and no requirement for installing updates in future.

voice-notes PWA

voice-notes as a pwa is exactly similar as the voice-notes webapp but with a significant distinction being that, voice-notes's pwa is a installable application available on android and windows systems using the same codebase and no requirement for installing updates in future.
